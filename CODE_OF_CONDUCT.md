
# Code of Conduct

## Our Commitment

The TargetJS community is committed to providing a welcoming, respectful, and inclusive environment for everyone.

We welcome contributors of all experience levels and encourage constructive discussions focused on improving the project.

## Expected Behavior

Please:

- Be respectful and courteous.
- Be constructive when giving feedback.
- Focus discussions on ideas and code rather than individuals.
- Welcome newcomers and help them learn.
- Assume good intentions whenever possible.

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment or personal attacks
- Discrimination or hateful language
- Threats or intimidation
- Trolling or intentionally disruptive behavior
- Publishing someone else's private information without permission
- Any behavior that creates an unwelcoming environment

## Scope

This Code of Conduct applies to all TargetJS community spaces, including:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Project-related communication

## Enforcement

Project maintainers may edit or remove comments, issues, pull requests, or other contributions that violate this Code of Conduct.

Repeated or serious violations may result in temporary or permanent removal from the project community.
